# 斯年智驾——中控平台

### 本地开发

- 安装依赖

```
yarn install
```

- 启动开发模式

```
yarn run serve
```

- 打包构建

```
yarn run build
```

- 格式化代码

```
yarn run lint
```

### 构建时环境变量

- NODE_ENV=production
- VUE_APP_PREVIEW=false
- VUE_APP_API_BASE_URL=/api
- BASE_URL=http://localhost:8000

### 运行时环境变量

- VUE_APP_API_ENDPOINT=http://localhost:8080/
